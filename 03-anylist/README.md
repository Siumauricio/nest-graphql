# Dev

1. Clone the repo
2. Copy the ```.env.example``` file to ```.env``` and fill in the values
3. Run ```yarn install```
4. Up the database with ```docker-compose up -d```
5. Run ```yarn start:dev```
6. Go to ```http://localhost:3000/graphql``` to access the GraphQL playground
